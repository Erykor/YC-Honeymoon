# Y♡C · 2026 欧洲蜜月

> **94 天 · 13 站 · 10 国 · 29 个核心节日/活动 · 2 人 · ~$39,400**
>
> 从大西洋海岸的奔马, 到皮埃蒙特的白松露; 从希腊海蚀岩壁的攀岩, 到地中海石灰岩 50m 大壁的潜水; 从维也纳金色大厅的华尔兹, 到克罗地亚 89 个荒岛的桨划; 从西西里浅滩的风筝水翼, 到 Chefchaouen 蓝城与 Marrakech medina 的香料巷。

**起点:** 7/29 北京/上海 → Dublin (经赫尔辛基/AMS/IST 12-15h)
**终点:** 10/30 Milan Malpensa → 北京/上海

**地理环线:** Ireland → UK → 中欧 (BUD→VIE 火车 → INN 火车) → 北意 Alpes → Adriatic 克罗地亚 → 希腊 Aegean → 西西里 → Morocco (PMO→RAK 飞 / 7 日文化环) → Malta → 中意 Tuscany → 法国 Côte d'Azur → 北意 Piemonte (Grasse→Alba 自驾 5h 沿法意边境) → Milan 返航。

**沉浸优先于打卡:** 6 个多日参与式 camp 是核心仪式 — Connemara 海岸骑乘 6 日 / Vienna Tanzschule Elmayer 华尔兹 3 日 / Kornati 海洋皮划艇 6 日 / Kalymnos 攀岩 7 日 / Lo Stagnone 风筝 14 日 deep / Malta PADI Advanced 3 日。Morocco 7 日文化环 (Marrakech + Fes + Chefchaouen) 是单独的多城慢游章节。
